Responsive Design
Digunakan agar website menjadi responsive 

sm	640px		@media (min-width: 640px) { ... }
md	768px		@media (min-width: 768px) { ... }
lg	1024px		@media (min-width: 1024px) { ... }
xl	1280px		@media (min-width: 1280px) { ... }
2xl	1536px		@media (min-width: 1536px) { ... }

contoh pengunaan :

<div class="sm:text-xl text-lg">
	<h1> Hello tailwind </h1> 
	</img src="logo" alt="logo">
</div>


maka text akan berukuran xl pada ukuran website lebih dari 640px, dan pada ukuran dibawah 640px (smartphone) maka akan berukuran lg 
jika ingin membuat tampilan pada smartphone maka harus dikasih default atau jangan dikasih embel embel responsive seperti sm: md: dll 
karena tailwind menyaraknkan untuk membuat tampilan mobile terlebih dahulu.
